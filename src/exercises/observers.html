Observers

Now that we've got our Observable and we know how to Subscribe with Observer and manage Subscriptions, it's easy to think about how we can have multiple subscriptions.

subscribe(btn: HTMLButtonElement) {
    const source = fromEvent(btn, "click").map(
        (event: Event) => `Event time: ${event.timeStamp}`
    );
    this.subscription = source.subscribe(new OutputObserver(this.output));
    this.subscription2 = source.subscribe(
        new ColoredOutputObserver(this.output2)
    );
}

unsubscribe() {
    this.subscription.unsubscribe();
    this.subscription2.unsubscribe();
}

When you click initiate, the output will appear in multiple destinations due to multiple subscriptions.
