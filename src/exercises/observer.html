Observers

The Subscriptions page showed how to create a subscription using an inlined observer definition. This page uses an custom class to contain the Observer logic.

class ConsoleObserver implements Observer {
    next(value: string) {
        console.info("Clicked: ", value);
    }
    error(err: any) {}
    complete() {}
}   


subscribe(btn: HTMLButtonElement) {
    const source = fromEvent(btn, "click").map(
        (event: Event) => `Event time: ${event.timeStamp}`
    );
    this.subscription = source.subscribe(new ConsoleObserver());
}

Click subscribe/unsubscribe to toggle the subscription. Look at what happens when you click the Initiate button when you are subscribed and unsubscribed.
