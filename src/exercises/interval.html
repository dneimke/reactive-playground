Observable Creation

In the previous example, we created an Observable based on a range. We saw that the numbers 1..100 appeared on the page as soon as we clicked Initiate.

In this demo we will use the interval operator to create our Observable. This will emit events at a given interval which gives the appearance of items being added gradually rather than all 100 items appearing instantaneously. The ReactiveX operators page lists the various creation operators. 

const source = Rx.Observable.interval(50)
        .take(100)
        .map(e => e + 1);

source.subscribe({
    next(value: number) {
        output.innerText += value === 100 ? value : `${value},`;

        if (value % 25 === 0) {
            output.innerHTML += `<br />`;
        }
    }
});

When you click initiate, the output will fill with numbers from a sequence using the range() operator.
