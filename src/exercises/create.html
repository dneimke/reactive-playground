Observable Creation

We now know a bit more about Observers, but what about Observables, where do they come and can we create them? 

const source = fromEvent(btn, "click");

...

const source = range(1, 100);

source.subscribe({
    next(value: number) {
        output.innerText += value === 100 ? value : `${value},`;

        if (value % 25 === 0) {
            output.innerHTML += `<br />`;
        }
    }
});

When you click initiate, the output will fill with numbers from a sequence using the range() operator.
